Redirects non-logged-in users to a custom login page before accessing checkout. Displays a message to log in when users land on the login page.

== Description ==
Redirect to Login for Checkout is a WooCommerce-compatible plugin that ensures only logged-in users can access checkout. Perfect for eCommerce websites, it allows you to set a custom login page, redirect users attempting checkout, and display a friendly popup message urging them to log in.

**Features:**
- WooCommerce compatibility
- Custom login page configuration
- SweetAlert-powered popup messages
- Simple settings page
- Lightweight and secure

For more information, visit [Netfie](https://www.netfie.com).

== Installation ==
1. Upload the `redirect-login-checkout` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin via the **Plugins** menu in WordPress.
3. Go to **Settings -> Redirect Login Settings** and configure the login page URL.

== Screenshots ==
1. **Settings Page**: Set your custom login page URL in the plugin's settings.
2. **Login Popup**: Users redirected to your login page will see a friendly message.
